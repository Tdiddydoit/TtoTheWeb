CSS History and Information

CSS Defintion

CSS stands for Cascading Style Sheets. CSS makes the HTML pages look good with layout, colors, font - basically any element on the page can be changed.

History of CSS

  • CSS 1 was releassed in 1996
  • CSS 2 was relased in 1998
  • CSS 3 (current version) - it is a living standard

Basic structure of CSS

selector{ property: value; propety2: value; }

  • CSS selectors can be an HTML element (ex: p, body)
  • CSS helps with the seperation of contnet and looks
  • Easy to maintain
  • Accessibility issues can be more easily managed

Three ways to inculde CSS on a webpage

  • External CSS in a seperate file
  • Embeded CSS is coded right into file
  • Inline applies to specific HTML element
